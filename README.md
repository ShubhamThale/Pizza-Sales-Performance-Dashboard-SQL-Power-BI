# Pizza Sales Performance Dashboard (Power BI)

## Project Overview
This project presents an interactive Pizza Sales Performance Dashboard developed using Microsoft Power BI. The dashboard provides insights into sales performance, customer ordering behavior, product popularity, and revenue trends.
---

## Business Problem
Pizza businesses generate large volumes of sales data. This dashboard helps stakeholders track revenue, identify top-selling products, analyze customer preferences, and make data-driven business decisions.
---

## Dashboard Features

### KPI Metrics
* Total Revenue
* Total Orders
* Total Pizzas Sold
* Average Order Value
* Average Pizzas per Order

### Sales Analysis
* Monthly Revenue Trend
* Orders by Day of Week
* Revenue by Pizza Category

### Product Analysis
* Top Pizza by Revenue
* Top Pizza by Orders
* Top Pizza by Quantity Sold
* Worst Performing Pizzas

### Category & Size Analysis
* Revenue by Pizza Category
* Sales Distribution by Pizza Size
* Pizza Sold by Category and Size

### Interactive Filters
* Month Filter
* Pizza Size Filter
* Pizza Category Filter
---

---

## Key Insights
* Total Revenue: $817.62K
* Total Orders: 21,334
* Total Pizzas Sold: 49,559
* Average Order Value: $38.32
* Classic category generates the highest revenue.
* Large-size pizzas contribute the highest share of sales.
* The Classic Deluxe Pizza is the top-performing pizza by orders and quantity sold.
